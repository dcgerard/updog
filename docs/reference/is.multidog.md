# Tests if an argument is a `multidog` object.

Tests if an argument is a `multidog` object.

## Usage

``` r
is.multidog(x)
```

## Arguments

- x:

  Anything.

## Value

A logical. `TRUE` if `x` is a `multidog` object, and `FALSE` otherwise.

## Author

David Gerard

## Examples

``` r
is.multidog("anything")
#> [1] FALSE
# FALSE
```
