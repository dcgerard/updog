# Tests if an argument is a `flexdog` object.

Tests if an argument is a `flexdog` object.

## Usage

``` r
is.flexdog(x)
```

## Arguments

- x:

  Anything.

## Value

A logical. `TRUE` if `x` is a `flexdog` object, and `FALSE` otherwise.

## Author

David Gerard

## Examples

``` r
is.flexdog("anything")
#> [1] FALSE
# FALSE
```
